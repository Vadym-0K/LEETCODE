# 22. Generate Parentheses


Given ```n``` pairs of parentheses, write a function to <i>generate all combinations of well-formed parentheses.</i>


<be>

***Example 1:***

Input: n = 3

Output: ["((()))","(()())","(())()","()(())","()()()"]

<br>

***Example 2:***

Input: n = 1
Output: ["()"]

<br>

***Constraints:***

```1 <= n <= 8```
